#  Duck Track - Back-End

  

Com ExpressoTS e MongoDB

  

##  Instalação

  

```bash

npm  run  install

```

  

##  Executando a aplicação

  

### Para subir o container use:

 
```bash
# Docker

docker-compose  up  -d

 ```
### Para executar o projeto use:
```bash

# Desenvolvimento

npm run start

  

# Build

npm run build

  

# Modo de produção

npm run start:prod

```
