# DuckTrack

## Installation

```bash
npm run install
```

## Running the app

```bash
# development
npm run start

# build
npm run build

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
npm run test

# test coverage
npm run test:cov
```
